# React-ROR-login-app

This app let's you login with google account.
Can be extended to database authentication as well.


### Instructions

+ Fork and clone the repository
+ Install ruby 3.1.0
+ Install esbuild for compiling react
+ Install yarn
+ run bundle install
+ run bin/dev
